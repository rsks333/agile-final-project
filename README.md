# Agile Final Project: Product Development with GitHub --

This project demonstrates the application of Agile methodologies using GitHub Projects.

## Features
* **Kanban Board**: Managed 10 user stories through Backlog, In Progress, and Done.
* **Agile Estimation**: Used Story Points (Size) for sprint planning.
* **Sprint Management**: Organized work into a time-bound Sprint milestone.
* **Templates**: Implemented an automated User Story template with Acceptance Criteria.
* **Analytics**: Tracked progress using Burn-up and Burndown visualizations.
